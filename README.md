<h2>🛠️ Installation Steps:</h2>

<p>1. Install the repo in your machine</p>

`git clone https://github.com/FavioMagallanes/weather-forecast.git`

<p>2. Move to the project folder</p>

`cd weather-app`

<p>3. Run</p>

`yarn`

<p>4. Finally, run</p>

`yarn start`
